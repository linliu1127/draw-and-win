# 這是一個類似麻將的撲克牌遊戲，規則如下：
1. 共有4名玩家，每人發4張牌。
2. 輪到你時，抽牌庫中的一張牌或是撿上一家丟的牌(連點兩下即可抽牌)。
3. 檢查手中的牌，棄掉一張牌使手牌保持在4張牌(連點兩下即可棄牌)。
4. 輪下一位玩家進行步驟2與3。
5. 持續進行直到有人胡牌。
6. 胡牌：胡牌條件為湊齊五張顏色一樣的牌(全黑或全紅)，並有一對以及一順。例如(2,3,4,10,10)。
7. 聽牌：聽牌時若有任一玩家(不一定是上家)打出所聽的牌就可以立即胡牌。
8. 自摸：聽牌時可以選擇不胡別人的牌等待輪到自己抽牌，若因此胡牌成功就叫自摸。
9. 魃：牌庫中有兩張鬼牌，稱作「魃」。魃可當作任意一張牌使用。
10. 算分方式：自摸勝利者向每位玩家收取100點；胡牌勝利者向被胡牌者收取50點。
11. 結束方式：每人起始1000點，玩到有人破產或是四人都同意不想玩為止。
