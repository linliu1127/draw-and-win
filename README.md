# 這是一個類似麻將的撲克牌遊戲，規則如下：
1. 共有4名玩家，每人發4張牌。
2. 遊戲目標是湊齊五張顏色一樣的牌(全黑或全紅)，且含一對以及一順。例如(2,3,4,10,10)。
3. 輪到你時，摸牌庫一張牌或是撿上家丟的牌(連點兩下即可抽牌)，若不構成自摸就棄一張牌換下一位玩家。
4. 若有任一玩家(不一定是上家)打出你聽的牌可以立即胡牌。但也可以放棄，等待自摸。
5. 自摸勝利者向每位玩家收取100點；胡牌勝利者向被胡牌者收取50點。下一局將由勝者先開始。
6. 魃：牌庫中有兩張鬼牌，稱作「魃」。魃可當作任意一張牌使用。
7. 結束方式：每人起始1000點，玩到有人破產或是四人都同意不想玩為止。
